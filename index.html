2x2 Terminal App...

Pursuing this project was what originally got me into coding! I'd been working on an academic musical project examining a class of musical object known as canonic sequences. This was a tool I wrote in the C language to generate a specific subset of these patterns and process them, and create formatted LaTeX files containing data on each pattern along with musical examples algorithmically typeset in LilyPond.

The topic is very esoteric, so it's hard to convey how much work was involved, or what it all means, but pursuing this project taught me a lot about algorithms and is really what made me fall in love with programming! üòÅ

2x2 Web App...

This project combined the DOM-manipulation techniques I learned from making this project gallery with my previous, terminal-based program to make a fully interactive web app to explore 2x2 canonic sequences. It will hopefully make a nice educational resource for musicians with an interest in theory.

The set of all 2x2 canonic sequences is generated algorithmically in JavaScript, and populates a column to the left-hand side of the screen. To the right, details about individual patterns the user clicks on are displayed, and above is a dynamic menu by which they can be filtered. Information about each pattern is also dynamically generated via various algorithms, and even the images were originally compiled from algorithmically generated LilyPond code, before being replaced with my own custom canvas-based notation renderer. I also added dynamically-generated audio using the Web Audio API.

Movie Data Project...

This page stores a collection of movie data as a JavaScript object and allows the user to view the data, edit individual movies, delete them, or add their own entries. It was a fun exercise in DOM manipulation and interacting with objects in JavaScript!

Snake Game...

Recreating the classic game "Snake" in JavaScript was a lot of fun! I learned how to draw onto a canvas element with JavaScript, how to continually redraw objects to create animation, and how to structure a very simple game in general.
